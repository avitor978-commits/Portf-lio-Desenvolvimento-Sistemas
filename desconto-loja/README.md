# Sistema de Desconto para Loja

Exercício de lógica de programação desenvolvido em Portugol Studio.

## Objetivo
Simular um sistema de vendas que:
- armazena o desconto do dia
- calcula desconto para compras acima de R$100
- exibe dados do cliente
- gera um relatório simples de vendas

## Conceitos aplicados
- Porcentagem
- Estrutura de repetição (`para`)
- Vetores
- Condicionais (`se`, `senão se`)

Este exercício foi **exigido pelo curso**.
