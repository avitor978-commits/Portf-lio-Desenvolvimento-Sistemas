# Cardápio Inteligente

Exercício de lógica de programação desenvolvido em Portugol Studio.

## Objetivo
Simular um sistema de cardápio que sugere pratos com base nas respostas do
cliente, considerando se ele:
- está de dieta
- é vegetariano

## Conceitos aplicados
- Estruturas condicionais (`se`, `senão se`, `senão`)
- Entrada e saída de dados
- Tomada de decisão lógica

Este exercício foi **exigido pelo curso**.
