# Projeto Integrador I – Sistema de Controle de Acesso de Portaria

Este diretório contém a documentação da **Etapa 1 – Levantamento e
especificação de requisitos**, desenvolvida no curso Técnico em
Desenvolvimento de Sistemas (SENAC).

O projeto encontra-se em andamento e será evoluído nas próximas etapas,
com modelagem de banco de dados e implementação em Java e MySQL.
